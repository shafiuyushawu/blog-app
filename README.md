> * Services (job queues, cache servers, search engines, etc.)

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need: any web-browser.

### Setup

Clone this [repository](https://github.com/shafiuyushawu/blog-app.git) to your desired folder.

### Install

This project does not require installation.

### Usage

**To run the project follow the following instruction:-**

- Clone this [repository](https://github.com/shafiuyushawu/blog-app.git) in your local machine with correct branch.
- Run following cmds:-
- **rails db:create**
- **rails db:migrate**
- **rails s**

### Run tests

To run tests, run the following command: **rspec**.
